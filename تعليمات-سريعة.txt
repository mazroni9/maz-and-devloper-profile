1️⃣  إنشاء المستودع على GitHub:
   - اضغط على الزر الأخضر "New" في GitHub (أعلى يسار الصفحة)
   - أو اذهب مباشرة إلى: https://github.com/new
   
   - اسم المستودع: dasm-e-team-profile
   - اختر Public أو Private
   - ⚠️ مهم: لا تضع علامة على:
     ☐ Add a README file
     ☐ Add .gitignore
     ☐ Choose a license
   
   - اضغط "Create repository"

2️⃣  بعد إنشاء المستودع، ارجع هنا واضغط Enter للمتابعة...
