  مخطط المعمارية - DASM-e Platform
═══════════════════════════════════════════════════════════════

الطبقة 1: المدخل (Frontend Layer)
─────────────────────────────────
👤 المستخدمون (تجار - أفراد - إدارة)
    ↓
💻 واجهة الويب (Next.js / Vercel)
    ↓
🛡️ بوابة الخدمات (API Gateway)

───────────────────────────────────────────────────────────────

الطبقة 2: الخدمات الأساسية (Core Services)
────────────────────────────────────────────
من بوابة الخدمات → 
    ├─ 🆔 الهوية (Identity)
    ├─ 🔨 المزادات (Auctions)
    ├─ 📋 العروض (Listings)
    ├─ 💰 المحفظة (Wallet)
    └─ 💳 المدفوعات (Payments)

───────────────────────────────────────────────────────────────

الطبقة 3: الخدمات المتخصصة (Domain Services)
──────────────────────────────────────────────
من الخدمات الأساسية:
    • 🔨 المزادات → ⚡ المزايدة اللحظية (Realtime)
    • 💻 واجهة الويب → ⚡ المزايدة اللحظية (Realtime)
    • 📋 العروض → 🚛 الأسطول (Fleet)
    • 🔨 المزادات → 🔔 الإشعارات (Notifications)
    • 💳 المدفوعات → 🔔 الإشعارات (Notifications)
    • 🚛 الأسطول → 🔔 الإشعارات (Notifications)
    • 🔨 المزادات → 🚨 الحماية (Fraud & Risk)
    • 💳 المدفوعات → 🚨 الحماية (Fraud & Risk)

───────────────────────────────────────────────────────────────

الطبقة 4: حافلة الرسائل (Message Bus)
──────────────────────────────────────
📨 حافلة الرسائل (Kafka / RabbitMQ)

من:
    • 🔨 المزادات
    • ⚡ المزايدة اللحظية
    • 💰 المحفظة
    • 💳 المدفوعات
    • 🚛 الأسطول
    • 🔔 الإشعارات
    • 🚨 الحماية

إلى:
    • 📊 التقارير (Analytics)
    • 🤖 الذكاء الاصطناعي (AI Insights)

───────────────────────────────────────────────────────────────

الطبقة 5: قواعد البيانات (Databases)
─────────────────────────────────────
كل خدمة لها قاعدة بيانات مستقلة:

    🆔 الهوية → DB الهوية
    🔨 المزادات → DB المزادات
    📋 العروض → DB العروض
    💰 المحفظة → DB المحفظة
    💳 المدفوعات → DB المدفوعات
    🚛 الأسطول → DB الأسطول
    📊 التقارير → DB التقارير

═══════════════════════════════════════════════════════════════
  مفتاح الألوان:
═══════════════════════════════════════════════════════════════

🔵 أزرق فاتح: واجهة المستخدم
🔵 أزرق: بوابة الخدمات
🟢 أخضر: الخدمات الأساسية (Core Services)
🟠 برتقالي: الخدمات المتخصصة (Domain Services)
🟣 بنفسجي: حافلة الرسائل (Message Bus)
⚫ رمادي: قواعد البيانات (Databases)
